	以下の仕様を満たす Main クラスを作ってください。
	
	🎯 仕様
	メニュー表示
	=== Book Library ===
	1. Add Book
	2. List Books
	3. Delete Book
	4. Search Book
	5. Exit
	> 
	
	📌 各機能の仕様
	1. Add Book
	
	タイトル入力
	
	著者入力
	
	nextId を使って Book を作成
	
	repo.add(book)
	
	「Added!」と表示
	
	2. List Books
	
	repo.findAll() を使う
	
	書籍を以下の形式で表示する：
	
	[ID] タイトル - 著者
	
	3. Delete Book
	
	削除したい ID を入力
	
	repo.delete(id) を呼ぶ
	
	戻り値が true → “Deleted”, false → “Not found”**
	
	4. Search Book
	
	検索キーワードを入力
	
	repo.searchByTitle(keyword) を使って検索
	
	0件 → "No books found."
	
	ヒット → list と同じ形式で表示
	
	5. Exit
	
	「Bye!」を表示して終了（return で OK）
