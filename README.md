# java_study

このリポジトリは、Java の学習用プロジェクトです。  
テキストに沿って作成した複数の小さなアプリケーションが含まれています。

---

## 📁 プロジェクト構成

`src/` 配下に以下のパッケージがあります。

### 1. **todo_app**
コンソールで動く Todo アプリ。  
- タスクの追加  
- タスクの一覧表示  
- タスクの削除  

Java の基本的なクラス設計・リスト操作・ID 管理などを学習するためのアプリです。

---

### 2. **memoApp**
メモを追加・一覧表示できる簡易メモアプリ。  
- メモ内容の入力  
- メモの保存  
- メモ一覧の表示

リポジトリパターンや ArrayList の操作を練習するためのアプリです。

---

### 3. **calculator**
簡単な四則演算をコンソールで実行できるアプリ。  
- 足し算  
- 引き算  
- 掛け算  
- 割り算  

Java の基本的な入力処理・条件分岐を練習する目的で作成しています。

---

### 4. **phonebook_app**
電話帳アプリ。  
- 連絡先の追加  
- 一覧表示  
- 名前で検索  
- ID 指定で削除  

クラスの設計、インスタンス管理、リポジトリの実装などを練習するアプリです。

---

## 🛠 使用技術
- **Java SE 21**
- **Eclipse IDE**
- 標準入力（Scanner）
- ArrayList を中心とした基本的なコレクション操作

---
