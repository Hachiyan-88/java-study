# java_study

このリポジトリは、Java の学習用プロジェクトです。  
テキストに沿って作成した複数の小さなアプリケーションが含まれています。

---

## 📁 プロジェクト構成

`src/` 配下に以下のパッケージがあります。

---

### 1. **todo_app**
コンソールで動く Todo アプリ。  
- タスクの追加  
- タスクの一覧表示  
- タスクの削除  

Java の基本的なクラス設計・リスト操作・ID 管理などを学習するためのアプリです。

---

### 2. **memoApp**
メモを追加・一覧表示できる簡易メモアプリ。  
- メモ内容の入力  
- メモの保存  
- メモ一覧の表示  

リポジトリパターンや ArrayList の操作を練習するためのアプリです。

---

### 3. **calculator**
簡単な四則演算をコンソールで実行できるアプリ。  
- 足し算  
- 引き算  
- 掛け算  
- 割り算  

Java の基本的な入力処理・条件分岐を練習する目的で作成しています。

---

### 4. **phonebook_app**
電話帳アプリ。  
- 連絡先の追加  
- 一覧表示  
- 名前で検索（部分一致）  
- ID 指定で削除  
- 検索結果の複数件対応  

クラス設計、インスタンス管理、リポジトリ実装、リスト操作の理解を深めるためのアプリです。

---

### 5. **bookLibrary_app**
書籍を管理するシンプルな図書アプリ。  
- 本の追加  
- 一覧表示  
- ID による削除  
- タイトル部分一致による検索  

Java でのデータ管理、クラスとリポジトリ構造、コレクション操作の応用練習として作成しています。

---

## 🛠 使用技術
- **Java SE 21**
- **Eclipse IDE**
- 標準入力（Scanner）
- ArrayList を中心とした基本的なコレクション操作

---
